# The Cataclysm SMP
**Cataclysm** es un proyecto de Minecraft inspirado en la modalidad *Permadeath* de **ElRichMC**, diseñado para ofrecer una experiencia de supervivencia EXTREMA.

---

## 🎮 Características
- **Modo hardcore**: Una sola vida.
- **Duración definida**: 35 días de contenido de dificultad progresiva.
