# The Cataclysm SMP

**Cataclysm** es un proyecto de Minecraft inspirado en la modalidad *Permadeath* creada por **ElRichMC** con el objetivo de otorgar una experiencia extremadamente dificil.

---

## 🎮 Características principales
- **Supervivencia hardcore**: Una sola vida, decisiones críticas y consecuencias permanentes.
- **Duración limitada**: El servidor se estructura en temporadas de 35 días.
- **Narrativa inmersiva**: Temática de apocalipsis con lore progresivo y eventos en vivo.
- **Comunidad activa**: Integración con Discord para coordinación, aplicación de jugadores y comunicación directa.
- 
---

## 🚀 Objetivos del proyecto
- Ofrecer una experiencia de juego única, basada en la tensión constante y la narrativa colaborativa.
- Construir una comunidad sólida de jugadores y creadores de contenido.
- Experimentar con mecánicas de *roleplay* y supervivencia en un entorno controlado.

---

## 📦 Instalación y acceso
1. **Requisitos previos:** Minecraft Java Edition (versión recomendada a definir según temporada).
2. **Aplicación:** Únete al [Discord oficial](#) y completa el formulario de aplicación.
3. **Conexión:** Una vez aceptado, recibirás la IP del servidor y las reglas oficiales.
